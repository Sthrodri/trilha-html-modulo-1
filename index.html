Linguagem de programação

(voltar)

A linguagem de programação é como uma "receita" que você escreve para o computador. Ela usa palavras e símbolos específicos que só a máquina entende. Com essa receita, você diz ao computador, passo a passo, o que ele deve fazer para criar um programa, um aplicativo ou um site.


HTML não é uma linguagem de programação?

(voltar)

O HTML não é uma linguagem de programação porque não é possivel dizer ao computador o que ele deve fazer. Ele é uma "linguagem de marcação", ou seja, ele serve para organizar o texto e o conteúdo de uma página para que o navegador consiga entender e mostrar na tela.

Estrutura básica de um HTML

(voltar)

A estrutura básica de um documento HTML é composta por elementos que definem a página. Os elementos mais comuns incluem:

  • <html>: Define o início e o fim de um documento HTML.
  • <head>: Contém informações sobre o documento, como o título e links para estilos.
  • <body>: Contém o conteúdo visível da página, como texto, imagens e links.

Falando sobre tags HTML

(voltar)

As tags HTML são elementos que definem a estrutura e o conteúdo de uma página web. Cada tag tem uma função específica e pode conter texto, imagens, links e outros elementos. Algumas das tags mais comuns incluem:

  • <p>: Define um parágrafo de texto.
  • <a>: Define um link para outra página ou recurso.
  • <img>: Define uma imagem a ser exibida na página.

Textos

(voltar)

Os textos em HTML são definidos principalmente pela tag <p>, que cria parágrafos. Além disso, você pode usar outras tags para formatar o texto, como <strong> para negrito, <i> para itálico e <u> para sublinhado.
